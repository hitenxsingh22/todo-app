# TODO Application

## Description
This is a simple TODO application built using NEXT for the frontend and Node.js with MongoDB as the backend. The application allows users to create, update, delete, and search for TODO items.

## Technologies Used
- Next
- Node.js
- Express
- MongoDB
- Tailwind CSS for styling
- React Icons for lightweight icons

## Installation

### Prerequisites
Make sure you have the following installed:
- Node.js
- npm (Node Package Manager)
- MongoDB (running locally or through a cloud service)

### Step-by-Step Setup

1. **Clone the Repository**

2. **Navigate to the Project Directory**


3. **Install Backend Dependencies**
Navigate to the backend folder:
   cd backend
   npm install

4. **Install Frontend Dependencies**
Navigate back to the root directory and then to the frontend folder:
   cd ..
   cd frontend
   npm install

5. **Setup Environment Variables**
Create a `.env` file in your backend folder with your MongoDB connection string

6. **Run the Backend Server**
Navigate to the backend directory and start the server:
   cd backend
   node server.js
By default, the backend should run on http://localhost:5000.

7. **Run the Frontend Application**
Open a new terminal, navigate to the frontend directory and run:
   cd frontend
   npm start

The application should launch on http://localhost:3000
